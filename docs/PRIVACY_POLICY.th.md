# นโยบายความเป็นส่วนตัวสำหรับ Auto Bot Trader i18n (ABTPi18n)

**อัปเดตล่าสุด:**: 10 พฤศจิกายน 2025

ยินดีต้อนรับสู่ Auto Bot Trader i18n ("ABTPi18n", "เรา", "ของเรา") นโยบายความเป็นส่วนตัวนี้อธิบายว่าเรารวบรวม, ใช้, เปิดเผย และปกป้องข้อมูลของคุณอย่างไรเมื่อคุณใช้แพลตฟอร์มและบริการของเรา

การใช้บริการของเราหมายความว่าคุณตกลงที่จะรวบรวมและใช้ข้อมูลตามนโยบายนี้

## 1. ข้อมูลที่เรารวบรวม

### A. ข้อมูลส่วนบุคคล
เราอาจรวบรวมข้อมูลส่วนบุคคล เช่น:
* **ที่อยู่อีเมล:** เพื่อสร้างบัญชี, ใช้ในการเข้าสู่ระบบ และเพื่อการสื่อสาร
* **ชื่อผู้ใช้:** เพื่อระบุตัวตนของคุณภายในแพลตฟอร์ม

### B. Exchange API Keys (ข้อมูลสำคัญยิ่งยวด)
เพื่อให้สามารถให้บริการบอทเทรดได้ เราจำเป็นต้องให้คุณระบุ API Keys (API Key และ API Secret) จาก Exchange ภายนอก (เช่น Binance, Bybit)
* เราจะร้องขอสิทธิ์ (Permissions) เท่าที่จำเป็นในการส่งคำสั่งเทรดและอ่านข้อมูลบัญชี/ประวัติการเทรดเท่านั้น
* เรา **จะไม่** ร้องขอสิทธิ์ในการถอน (Withdrawal) ผ่าน API และเราขอแนะนำอย่างยิ่งให้คุณปิดใช้งานสิทธิ์นี้ที่ตัว API Key

### C. ข้อมูลที่รวบรวมโดยอัตโนมัติ
เราอาจรวบรวมข้อมูลเกี่ยวกับการใช้งานบริการของคุณโดยอัตโนมัติ:
* **ข้อมูล Log:** ที่อยู่ IP, ประเภทเบราว์เซอร์, หน้าที่เข้าชม และการประทับเวลา
* **ข้อมูลบอทและการเทรด:** การตั้งค่ากลยุทธ์, ประวัติการเทรด, ข้อมูล PnL และ Log ข้อผิดพลาดที่สร้างโดยบอทของคุณ

## 2. วิธีที่เราใช้ข้อมูลของคุณ

ข้อมูลของคุณจะถูกใช้เพื่อวัตถุประสงค์ดังต่อไปนี้:

* **เพื่อให้บริการและบำรุงรักษา:** เพื่อสร้างบัญชีของคุณ, สั่งการบอทเทรดของคุณ และแสดงผลแดชบอร์ด
* **เพื่อดำเนินการเทรด:** **Exchange API Keys** ของคุณจะถูกใช้ *เพื่อวัตถุประสงค์เดียว* คือการส่งคำสั่งเทรดและอ่านข้อมูลบัญชีจาก Exchange ในนามของคุณ ตามคำสั่งจากกลยุทธ์ที่คุณเลือก
* **เพื่อปรับปรุงบริการ:** เราวิเคราะห์ข้อมูลการใช้งานเพื่อระบุข้อบกพร่อง, ปรับปรุงประสิทธิภาพ และพัฒนาฟีเจอร์ใหม่
* **เพื่อสื่อสารกับคุณ:** เพื่อส่งการแจ้งเตือนด้านความปลอดภัย, การแจ้งเตือนเกี่ยวกับบัญชี และข้อความสนับสนุน

## 3. วิธีที่เราปกป้องข้อมูลของคุณ (Security First)

เราใช้มาตรการรักษาความปลอดภัยระดับ Omega-Tier เพื่อปกป้องข้อมูลของคุณ โดยเฉพาะ API Keys ของคุณ

* **การเข้ารหัสขณะจัดเก็บ (Encryption at Rest):** **API Secret ของคุณจะถูกเข้ารหัสภาคบังคับ** (เช่น ใช้ AES-GCM 256-bit) *ก่อน* ที่จะถูกจัดเก็บลงในฐานข้อมูลของเรา โดยจะถูกเก็บในรูปแบบที่เข้ารหัสแล้ว และคีย์ถอดรหัสจะถูกจัดการแยกต่างหากอย่างปลอดภัย
* **การเข้ารหัสขณะส่ง (Encryption in Transit):** การสื่อสารทั้งหมดระหว่างคุณ, เซิร์ฟเวอร์ของเรา และ Exchange จะถูกเข้ารหัสโดยใช้ SSL/TLS
* **การควบคุมการเข้าถึง:** เราบังคับใช้นโยบายการควบคุมการเข้าถึงภายในที่เข้มงวด มีเพียงระบบอัตโนมัติที่จำเป็น (เช่น Bot Runner) เท่านั้นที่สามารถขอถอดรหัสคีย์ของคุณเพื่อทำการเทรดได้ และการเข้าถึงนี้จะถูกบันทึกและตรวจสอบอย่างเข้มงวด
* **ความปลอดภัยฝั่งผู้ใช้:** เราขอแนะนำและกระตุ้นให้ผู้ใช้ทุกคน **ใช้ IP Whitelisting** บน Exchange API Keys ของตน โดยจำกัดการเข้าถึงให้เฉพาะ IP Address ที่กำหนดของแพลตฟอร์มเราเท่านั้น

## 4. การแบ่งปันและเปิดเผยข้อมูล

เรา **ไม่** ขาย, แลกเปลี่ยน, ให้เช่า หรือแบ่งปันข้อมูลส่วนบุคคลหรือ API Keys ของคุณกับบุคคลที่สามเพื่อวัตถุประสงค์ทางการตลาด

เราอาจเปิดเผยข้อมูลของคุณในกรณีจำกัดต่อไปนี้เท่านั้น:
* **เมื่อได้รับความยินยอมจากคุณ:** หากคุณอนุญาตให้เราทำเช่นนั้นอย่างชัดเจน
* **เพื่อปฏิบัติตามกฎหมาย:** หากกฎหมาย, หมายศาล หรือกระบวนการทางกฎหมายอื่น ๆ กำหนด
* **เพื่อบังคับใช้สิทธิ์ของเรา:** เพื่อปกป้องและปกป้องสิทธิ์, ทรัพย์สิน หรือความปลอดภัยของ ABTPi18n, ผู้ใช้ของเรา หรือสาธารณะ

## 5. การเก็บรักษาข้อมูล

เราจะเก็บรักษาข้อมูลส่วนบุคคลและ API Keys ของคุณตราบเท่าที่บัญชีของคุณยังคงใช้งานอยู่กับเรา หากคุณลบบัญชีของคุณ เราจะปฏิบัติตามกระบวนการลบข้อมูลของคุณ รวมถึง API Keys ที่เข้ารหัสและประวัติการเทรด ออกจากระบบการผลิตของเรา

## 6. สิทธิ์ของคุณ

คุณมีสิทธิ์ในการเข้าถึง, อัปเดต หรือลบข้อมูลส่วนบุคคลของคุณ คุณสามารถจัดการ API Keys ของคุณได้โดยตรงจากแผงการตั้งค่า รวมถึงการเพิ่มหรือเพิกถอนได้ตลอดเวลา

## 7. การเปลี่ยนแปลงนโยบายความเป็นส่วนตัวนี้

เราอาจปรับปรุงนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว เราจะแจ้งให้คุณทราบถึงการเปลี่ยนแปลงใด ๆ โดยการโพสต์นโยบายใหม่ในหน้านี้และอัปเดตวันที่ "อัปเดตล่าสุด"
